### Getting Started

#### Install

```yml
dependencies:
  tiengviet: ^0.4.1
```

You can install packages from the command line:

```bash
pub get
-- or --
flutter pub get
```

#### Usage

```dart
import 'package:tiengviet/tiengviet.dart';

print(TiengViet.parse('Xin chào việt nam')); // xin chao viet nam
```
